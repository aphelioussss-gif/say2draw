# DESIGN.md - Say2Draw 设计文档

## 1. 设计目标

创建一个直观、响应迅速的语音绘图体验，让用户专注于创意表达而非工具操作。

## 2. 交互设计

### 2.1 语音交互流程

```
用户说话 → 语音识别 → 文本转写 → LLM 解析 → 绘图指令 → Canvas 执行
```

### 2.2 反馈机制

- **视觉反馈**：语音识别状态指示器
- **听觉反馈**：操作确认音效
- **画布反馈**：实时绘图预览

### 3.2 状态指示

- 🎤 正在聆听（绿色脉冲）
- ⏳ 处理中（蓝色旋转）
- ✅ 操作完成（绿色对勾）
- ❌ 识别失败（红色提示）
